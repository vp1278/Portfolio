# Portfolio

## 1. Mapping and Detecting Ridges on Metal Coated Products
**BlueScope Steel | Jan 2025 – Sept 2025**

![Ridge Detection](images/ridge_detection.png)

### Objective
Developed a method to detect ridge formation during hot dip galvanising using XRF coating weight measurements.

Ridges are regions of excess metal coating that can deform coils and affect downstream processes such as painting and roll forming.

### Approach
- Extracted raw XRF data using signal processing techniques
- Applied smoothing and filtering methods to identify coating variations
- Developed an algorithm to automatically detect ridge peaks
- Validated predictions using defective coil samples

### Outcome
The detection method was validated and prepared for implementation on the production line.
